# Experiencia Inmersiva en Unreal Engine

## Proyecto realizado con unreal 5.5.4

> Configuracion para instalar (dentro de la carpeta de los proyectos de unreal)

git clone https://github.com/NIKO-WAFLLE/Experiencia_Inmersiva_en_Unreal_Engine.git

> acceder al repositorio clonado y instalar lo sigueinte

git lfs install

git lfs pull

> siempre antes de trabjar (copiar los cambios de los compañeros)

git pull

git lfs pull

> para subir cambios

git add Config Content *.uproject

git commit -m "mensaje claro"

git push

> descargar cambios cuando da error (borra cambios locales)

git restore .

git clean -fd

git fetch

git pull

git lfs pull
